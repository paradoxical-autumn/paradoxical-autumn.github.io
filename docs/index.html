Untitled Parkour Game

This was a prototype I created to test my skills within learning a new engine, being Flax Engine. I managed to get the initial prototype released for testing within a single week.

This prototype tested my skills to:

  • Learn how to create infinite level generation within a new engine
  • Learn an optimised way to locate and organise files.
